# JavaScript Practice: Easy, Medium, Hard

This repository is curated to help new JavaScript learners get meaningful repetition at multiple difficulty levels.

Many online resources provide only a few problems per concept and then expect full understanding. This repo is designed to solve that gap by giving you repeated concept practice in easy, medium, and hard variations.

## What This Repo Is For

- building confidence through repetition
- practicing the same concepts in different problem shapes
- progressing gradually from easy to medium to hard
- reducing the "I saw it once but still cannot apply it" problem

## Included Drill Families

## 1) Switch / Loops / Queues / Objects

Path:
- `switch-loops-queues-objects-drill-packs-easy-medium-hard/`

Practice focus:
- `switch` statements
- `while` and `do...while`
- queue-style processing
- object counting and key analysis

## 2) Bird Watcher: Arrays / For Loops

Path:
- `arrays-forloops-birdwatcher-drill-packs-easy-medium-hard/`

Practice focus:
- array indexing
- `for` loop patterns
- weekly aggregation logic
- streaks, rolling windows, and log correction

## Difficulty Structure

Each drill family includes:
- easy
- medium
- hard

## Running Tests

Each parent drill folder includes a root `package.json` with scripts such as:
- `npm run test:easy`
- `npm run test:medium`
- `npm run test:hard`
- `npm run test`

Run commands from the parent folder of the drill family you are practicing.
